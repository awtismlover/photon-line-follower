## Hardware

- uC - STM32G474RET6
- drivers - Custom ESCs
- sensor - 12x KTIR0711S
- battery - 300mAh 3s LiPo
- wheels - Kyosho Mini Z
- motors - 1404 BLDC
- Bluetooth - HC-04
